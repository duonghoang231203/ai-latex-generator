# Project Roadmap

## Phase 1: MVP (Completed)
- Document generation (various templates).
- LaTeX compilation using Tectonic sandbox.
- Auto-repair compilation loop.
- File-based CRUD operations.
- Chat-based iterative editing.

## Phase 2: Advanced Features (Upcoming)
- RAG (Retrieval-Augmented Generation) for incorporating user reference material.
- Markdown to LaTeX conversion.
- OCR for processing images/equations.
- Multi-file project support.
- Agentic multi-step document assembly.

## Phase 3: Platform Maturity
- Multi-user Authentication & Database integration (v2).
- Advanced deployment strategies.
