# Project Roadmap

## Phase 1: MVP (Completed)
- Document generation (various templates).
- LaTeX compilation using Tectonic sandbox.
- Auto-repair compilation loop.
- File-based CRUD operations.
- Chat-based iterative editing.

## Phase 2: Advanced Features (Upcoming)
1. **Multi-file project support (Core)**: Kiến trúc lưu trữ dạng thư mục (Directory-based storage) phục vụ tài liệu lớn.
2. **Agentic multi-step document assembly**: Cơ chế tạo dàn ý và tự động viết nội dung theo dạng Checklist (Human-in-the-loop).
3. **RAG (Retrieval-Augmented Generation)**: Tích hợp tài liệu tham khảo để tăng độ chính xác của nội dung sinh ra.
4. **OCR**: Xử lý và nhận diện công thức Toán/Lý/Hóa từ hình ảnh.
5. **Markdown to LaTeX conversion**: Viết nháp bằng Markdown, chuyển đổi tự động sang LaTeX.

## Phase 3: Platform Maturity
- Multi-user Authentication & Database integration (v2).
- Advanced deployment strategies.
